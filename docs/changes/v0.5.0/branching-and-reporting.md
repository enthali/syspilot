## Summary

Adds a **Branch step (Step 0)** to the Change Manager workflow so the CM creates
a `feature/<name>` branch from the latest feature branch before starting work
(chained branching strategy). Also adds a **Completion Reporting** rule to the
Orchestration Skill requiring managers to report back to the sender via
`jarvis_sendToSession` when a delegated task completes.

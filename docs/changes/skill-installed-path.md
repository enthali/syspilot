## Summary

The SKILL.md in `syspilot/skills/syspilot.impact-python/` is the product template,
installed by Setup Agent into `.github/skills/syspilot.impact-python/`. Copilot agents
read exclusively from `.github/` at runtime. Specification elements incorrectly referenced
the product source path (`syspilot/skills/...`) instead of the installed runtime path
(`.github/skills/...`). This change corrects all Design Spec references to use the path
visible to agents.
